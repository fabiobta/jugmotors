Aplicação feita no HandsOn de Groovy/Grails do JUGMS dia 17/09/2011. A idéia é ter uma aplicaçãozinha didática mesmo, nada muito pretencioso.

Algumas coisas presentes nesse projeto:

* Scaffolding estatico e dinâmico
* Validators
* Interceptors
* Filtros
* Ajax
* Content Negotiation
* Customização de templates
* UrlMapping

O login é estático: admin, admin
